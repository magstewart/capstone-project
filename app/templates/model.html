A Model to Predict the Probability of Human Cause

A soft classifier model was trained to predict the probability of human cause given a fire.

Using the predicted probability, we can calculate the expected return for the state as a result of investigating that fire. By ranking the wildfires according to this expected return, we can prioritze investigating the ones that are likely to bring in more money. The results can be deployed as a web-based dashboard.
